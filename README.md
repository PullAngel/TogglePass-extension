# TogglePass

TogglePass is a Chromium-based browser extension that allows users to temporarily toggle the visibility of password fields in a controlled, reversible, and non-invasive way.

## Status
🚧 MVP under active development.

## Supported Browsers
- Chrome
- Brave

## Project Structure
- `/docs` — project documentation
- `/src` — extension source code
- `/assets` — icons and static resources

## Security Note
TogglePass does not store, transmit, or persist sensitive data.
